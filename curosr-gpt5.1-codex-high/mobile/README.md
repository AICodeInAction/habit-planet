# Habit Planet · React Native Prototype

This Expo application mirrors the high-fidelity Habit Planet experience，提供全可交互界面，数据均来自内置 mockdata，方便产品、设计与研发快速验证。

## 运行方式

```bash
cd mobile
npm install        # 或 pnpm / yarn
npx expo start
```

- `expo start` 后可使用 iOS 模拟器、Android 模拟器或 Expo Go 真机调试。
- 所有接口均来自 `src/data/mockData.ts`，如需联调可在此替换为真实 API。
- React Navigation 的底部 Tab 完全可互动；各模块点击即可触发状态更新、点赞、推进星球或奖章进度。

## 主要结构

- `src/hooks/PrototypeProvider.tsx`：集中管理任务、家庭成员、星球与奖章的 mock 状态，提供交互函数。
- `src/screens/*`：对应首页、星际地图、家庭互动、奖章以及个人档案五大界面。
- `src/components/*`：复用型组件（任务卡片、星球卡片、圆形进度等），可直接在真实项目中引用。
