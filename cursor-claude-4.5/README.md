# 🌟 习惯星球 - Habit Planet

一个基于《掌控习惯》原理的亲子习惯养成应用高保真原型

## 📱 项目概述

习惯星球是一款专为儿童和家庭设计的习惯养成应用，通过星际探索的游戏化方式，让孩子在有趣的互动中培养良好习惯。

## 🎯 核心设计理念

基于《掌控习惯》（Atomic Habits）的四大定律：

1. **让它显而易见** - 清晰的习惯列表和直观的打卡界面
2. **让它有吸引力** - 星际探索主题、精美视觉设计、奖牌系统
3. **让它简便易行** - 一键打卡、简单操作流程
4. **让它令人愉悦** - 即时反馈、成就解锁、家庭互动

## 🌈 核心功能

### 1. 星际地图（home.html）
- 可视化的星球探索进度
- 通过习惯积累解锁新星球
- 实时显示今日进度和连续天数
- 动画效果增强沉浸感

### 2. 习惯打卡（habits.html）
- 按时段分类的习惯列表（早晨/下午/晚上）
- 一键打卡功能
- 连续打卡天数显示
- 日历选择查看历史记录
- 实时统计今日完成情况

### 3. 成就中心（achievements.html）
- 多层级奖牌系统
  - 传奇奖牌（连续打卡成就）
  - 星星收集者（积分成就）
  - 星际探险家（探索成就）
  - 家庭荣耀（互动成就）
- 进度展示和下一个目标提示
- 动画效果增强成就感

### 4. 家庭互动（family.html）
- 家庭成员展示和数据对比
- 家庭动态时间线
- 相互点赞鼓励机制
- 家庭挑战系统
- 家庭总成就统计

### 5. 个人中心（profile.html）
- 个人信息和等级系统
- 本周数据统计图表
- 目标管理
- 装饰物管理
- 设置和帮助中心

### 游戏化元素

1. 直接在浏览器中打开 `index.html` 即可查看所有界面的原型展示

## 🎯 用户体验分析

### 目标用户
- **主要用户**：6-12岁儿童
- **次要用户**：家长（监督和互动）

### 使用场景
1. 早晨起床后打卡晨间习惯
2. 放学后记录学习和运动习惯
3. 睡前完成晚间习惯打卡
4. 周末查看数据和解锁新星球
5. 家庭时间一起互动和鼓励

### 核心价值
- 让习惯养成变得有趣
- 增强家庭互动和亲子关系
- 培养孩子的自律能力
- 可视化进度增强成就感

## 📱 移动端开发

当前原型可以直接作为以下开发的参考：

- 使用 SwiftUI 实现

- 集成 Core Data 做本地存储
